# دليل الإعداد والتشغيل

## المتطلبات

- Node.js (الإصدار 18 أو أحدث)
- Backend API يعمل على المنفذ 5000

## خطوات التشغيل

### 1. تشغيل الـ Backend

تأكد من تشغيل الـ Backend API أولاً على المنفذ 5000

### 2. تشغيل الـ Frontend

```bash
cd bookstore-frontend
npm install
npm run dev
```

### 3. فتح المتصفح

افتح المتصفح على العنوان:
```
http://localhost:3000
```

## الصفحات المتاحة

1. **الرئيسية** (`/`) - لوحة التحكم مع الإحصائيات
2. **الكتب** (`/books`) - إدارة الكتب (عرض، إضافة، تعديل، حذف)
3. **العملاء** (`/customers`) - إدارة العملاء
4. **المبيعات** (`/sales`) - إنشاء عمليات بيع جديدة
5. **الإيجارات** (`/rentals`) - إنشاء عمليات إيجار جديدة
6. **التقارير** (`/reports`) - عرض التقارير والإحصائيات

## تغيير عنوان الـ API

إذا كان الـ Backend يعمل على منفذ مختلف، قم بتعديل الملف:
```
src/services/api.js
```

وغير السطر:
```javascript
const API_BASE_URL = 'http://localhost:5000/api'
```

## حل المشاكل الشائعة

### مشكلة الاتصال بالـ API

تأكد من:
- تشغيل الـ Backend
- عنوان الـ API صحيح
- لا يوجد Firewall يمنع الاتصال

### مشكلة CORS

إذا ظهرت مشكلة CORS، تأكد من إعدادات CORS في الـ Backend تسمح بالطلبات من `http://localhost:3000`

## البناء للإنتاج

```bash
npm run build
```

سيتم إنشاء ملفات الإنتاج في مجلد `dist`

لمعاينة البناء:
```bash
npm run preview
```
